Bounds on number of cuts


Recently there are some advances on counting the number of min-cut in graphs.

Consider we have an undirected graph G=(V,E)G=(V,E)G=(V,E) of nnn vertices, and there are positive cost c:E→R+c:E\to \R^+c:E→R+ on the edges. We define c(F)=∑e∈Fc(e)c(F)=\sum_{e\in F}c(e)c(F)=∑e∈F​c(e), to be the cost (value) of F⊂EF\subset EF⊂E.

Let P\mathcal{P}P be a partition of VVV where each partition class is non-empty. We define E(P)E(\mathcal{P})E(P) to be the set of edges with end points in two different partition classes. A set of edges FFF is called a kkk-cut, if F=E(P)F=E(\mathcal{P})F=E(P) for some P\mathcal{P}P such that ∣P∣≥k|\mathcal{P}|\geq k∣P∣≥k.

We stress that by this definition, a kkk-cut is a k−1k-1k−1-cut. A cut is a 222-cut. A min-kkk-cut is a kkk-cut of minimum value (cost). We let λk\lambda_kλk​ to denote the value of the min-kkk-cut.

It is well known that the number of min-cuts in a graph is (n2)=O(n2){n\choose 2} = O(n^2)(2n​)=O(n2). [1]

In the entire article, unless specifically stated, we assume kkk is a fixed integer at least 222, and α\alphaα is a fixed value at least 111.

We can express the state alternatively.

Theorem1

The number of cuts FFF such that c(F)≤λ2c(F)\leq \lambda_2c(F)≤λ2​ is O(n2)O(n^2)O(n2).

1 Bounds related to scaling of the min-cut

What happens when we want to know about the number of cuts with value at most αλ2\alpha \lambda_2αλ2​?

By simply analyzing Karger’s algorithm, one can obtain the following.

Theorem2

The number of cuts FFF such that c(F)≤αλ2c(F)\leq \alpha \lambda_2c(F)≤αλ2​ is O(n2α)O(n^{2\alpha})O(n2α).

With more careful analysis using tree packing, Karger obtained the following [2].

Theorem3

The number of cuts FFF such that c(F)≤αλ2c(F)\leq \alpha \lambda_2c(F)≤αλ2​ is O(n⌊2α⌋)O(n^{\floor{2\alpha}})O(n⌊2α⌋).

Indeed, we do have a lower bound of (n⌊2α⌋){n \choose \floor{2\alpha}}(⌊2α⌋n​). Just consider an unweighted cycle, where min-cut has value 222. We can pick any ⌊2α⌋\floor{2\alpha}⌊2α⌋ edges, which forms a cut of value at most α\alphaα times the min-cut.

Note that we require α\alphaα to a fixed value. This is because there is a dependency on α\alphaα hidden inside the big OOO. Our lower bound is absolute and does not depending on α\alphaα being fixed. It be interesting to obtain a matching upper bound. Hence we can consider the problem with strict inequality.

Conjecture4

The number of cuts FFF such that c(F)<αλ2c(F)< \alpha \lambda_2c(F)<αλ2​ is O(n⌈2α⌉−1)O(n^{\ceil{2\alpha}-1})O(n⌈2α⌉−1).

Henzinger and Williamson showed the conjecture true for all α≤32\alpha\leq \frac{3}{2}α≤23​ [3].

2 Bounds related to min-kkk-cut

There are multiple ways to obtain the following theorem. For example, directly generalize Karger’s argument for cut counting.

Theorem5

The number of cuts FFF such that c(F)≤λkc(F)\leq \lambda_kc(F)≤λk​ is O(n2(k−1))O(n^{2(k-1)})O(n2(k−1)).

There were many attempts, and people can only obtain lower bounds of the form (nk){n\choose k}(kn​). Again, a cycle would be an example of such lower bound. The min-kkk-cut has value kkk, and can be obtained by picking any kkk edges. The gap is pretty large. Hence one would tempt to conjecture the following.

Conjecture6

The number of cuts FFF such that c(F)≤λkc(F)\leq \lambda_kc(F)≤λk​ is O(nk)O(n^k)O(nk).

Recently, Gupta, Lee and Li almost closes the gap [4].

Theorem7

The number of cuts FFF such that c(F)≤λkc(F)\leq \lambda_kc(F)≤λk​ is O^(nk)\hat{O}(n^k)O^(nk).

Here O^\hat{O}O^ hides a factor smaller than any nϵn^\epsilonnϵ. While closing the gap, they showed thhe following interesting theorem.

Theorem8

The number of cuts FFF such that c(F)≤(2−ϵ)λkkc(F)\leq \frac{(2-\epsilon)\lambda_k}{k}c(F)≤k(2−ϵ)λk​​ is O(n)O(n)O(n).

Conjecture9

The number of cuts FFF such that c(F)<2λkkc(F)< \frac{2\lambda_k}{k}c(F)<k2λk​​ is O(n)O(n)O(n).

Note this theorem is basically shows we can also obtain interesting results for α=2−ϵk<1\alpha = \frac{2-\epsilon}{k} < 1α=k2−ϵ​<1.

How about approximate min-kkk-cuts? Chekuri, Quanrud and I extended the tree packing analysis of Karger, and obtained the following result for kkk-cuts [5].

Theorem10

The number of cuts FFF such that c(F)≤αλkc(F)\leq \alpha \lambda_kc(F)≤αλk​ is O(n⌊α2(k−1)⌋)O(n^{\floor{\alpha 2(k-1)}})O(n⌊α2(k−1)⌋).

Combining the Conjecture 6 and Conjecture 4, we get a unified conjecture, even for α<1\alpha<1α<1.

Conjecture11

The number of cuts FFF such that c(F)<αλkc(F)< \alpha \lambda_kc(F)<αλk​ is O(n⌈αk⌉−1)O(n^{\ceil{\alpha k}-1})O(n⌈αk⌉−1).
