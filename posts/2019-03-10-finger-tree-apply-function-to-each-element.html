Many problems actually require update to a entire interval of the sequence, which makes this extremely valuable. For example, consider the following simple problem.

Maintain a sequence of integers a1,…,ana_1,\ldots,a_na1​,…,an​, such that we can do the operation inc(i,j)inc(i,j)inc(i,j), which increment all numbers from iiith to jjjth index by 111. Also, it has a function value(i)value(i)value(i) which returns aia_iai​. This problem can be solved by finger tree with function update operation.
