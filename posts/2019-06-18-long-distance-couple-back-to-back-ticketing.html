Long distance couple back-to-back ticketing


The reason I explored an algorithm for covert back-to-back ticketing in the previous blog post, was because as someone currently in a long distance relationship. It is important to save money on travel.

For long distance couples, two people can fly. Consider if the couple decides be together each weekend, one can fly to the city of the other on Friday night, and comes back on Sunday night. Again we can take advantage of the back-to-back ticketing.

Let MMM be a matching of a graph GGG where the edges are two colored. MMM is called valid if vertex 2i2i2i and 2i−12i-12i−1 are contained in the same colored edge in MMM.

Problem1Couple back-to-back ticketing problem

Input: A multigraph G=(V,E)G=(V,E)G=(V,E) where V⊆[n]V\subseteq [n]V⊆[n], where each edge can be either red or blue, and there is an edge cost function c:E→R+c:E\to \R^+c:E→R+.

Output: A valid perfect matching MMM (allowing self-loops), such that the cost is minimized.

Unfortunately, I don’t see how to solve this problem in polynomial time. I could convert this problem to something that might be solved in polynomial time in the future.

Note we can split the vertices. That is, for vertex iii, we split it into (i,R)(i,R)(i,R) and (i,B)(i,B)(i,B) for its blue and red counter part. A matching is paired if (2i,C)(2i,C)(2i,C) and (2i−1,C)(2i-1,C)(2i−1,C) both has to be matched, where C∈{R,B}C\in \set{R,B}C∈{R,B}. The matching is split if (i,R)(i,R)(i,R) and (i,B)(i,B)(i,B) cannot be both in the matching.

If we just enforce one property – paired or split – then it is solvable in polynomial time. Indeed, both are matching under restrictions. We are interested in finding a matching MMM, such that the vertices covered by MMM is in some special structure. Namely Δ\DeltaΔ-matroid. Since the paired property is a very special Δ\DeltaΔ-matroid [1], which was first solved in [2]. The second property is partition matroid, which can be handled by hierarchical bbb-matching. Reader can see each individual restriction, the problem can be turned into a maximum weight perfect matching.

If we enforce both property, the problem is open. Although the structure is still very special, it is an even Δ\DeltaΔ-matroid. Naonori Kakimura communicated a more fundamental problem which can be solved by this problem.

Problem2Matching with disjoint pair constraints

Given a graph where the edges are partitioned into pairs. Find a maximum matching where it uses at most one edge of each pair.

I would be very interested to know the solution to that problem.
